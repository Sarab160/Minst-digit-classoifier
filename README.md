# MNIST Handwritten Digit Classification

This project uses Python and scikit-learn to classify handwritten digits from the MNIST dataset.

## Project Overview
- Loads the MNIST dataset using `fetch_openml`
- Visualizes sample digits using `matplotlib` and `seaborn`
- Preprocesses data with `StandardScaler`
- Trains a **Random Forest Classifier**
- Evaluates model performance using:
  - Accuracy
  - Precision
  - Recall
  - F1 Score
  - Confusion Matrix
- Visualizes predictions on random unseen images

## Features
- Data visualization of sample digits
- Model training and evaluation
- Confusion matrix heatmap
- Prediction visualization on unseen samples

## Requirements
- Python 3.x
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
